1-) Open Chrome or Firefox and Log in NovaRO Website, check 'Remember Me' (You can close the browser afterwards) 

2-) Enjoy the Program! =)

Notes:

Property --> None for No Property, otherwise copy Properties from NovaRO Website*
Medians --> Calculated based on your Refine (Limited to Property = None).
Notification --> Program creates an Task Bar icon for each Windows Notification triggered.
Cheap Column --> "Amount of Lowest Price Items" / "Amount of Items Below Median".   

Table sorted by Short Median.

Important: If your refine was not found, until it's found an above refine will be displayed. 
	   
Important 2: For Elements, write just the elements name (e.g: Water, Fire, Shadow...).
